# 术语表

| 术语 | 简要说明 |
|------|----------|
| 特征 | 描述样本的输入变量 |
| 标签 | 监督学习中要预测的目标 |
| 过拟合 / 欠拟合 | 训练好但泛化差 / 训练亦差 |
| 超参数 | 不由 `fit` 自动学习的配置项 |
| 估计器 | sklearn 中带 `fit` 的对象 |
| 交叉验证 | 多折训练与评估以降低划分偶然性 |
| Bagging / Boosting | 并行自助集成 / 序列加权集成 |

评估指标与 API 详见 [sklearn 指标](/foundations/sklearn/06-metrics)。
