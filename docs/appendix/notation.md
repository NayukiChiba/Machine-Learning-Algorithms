# 符号与记号

算法推导中常用符号如下；若某文有特殊约定，以该文为准。

## 线性代数

| 符号 | 含义 |
|------|------|
| $\mathbb{R}^d$ | $d$ 维实向量空间 |
| $\mathbf{x}, \mathbf{y}$ | 列向量 |
| $X$ | 设计矩阵（行常为样本） |
| $X^\top$ | 转置 |
| $\|\mathbf{x}\|$ | 范数（未注明时多为欧氏） |
| $\mathbb{1}(\cdot)$ | 指示函数 |

## 机器学习

| 符号 | 含义 |
|------|------|
| $n$ | 样本数；$d$ | 特征维数 |
| $y_i$ / $\hat{y}$ | 真值 / 预测 |
| $\mathcal{L}$ | 损失 |
| $\lambda, \alpha, C$ | 正则或 SVM 惩罚（具体见各算法文） |
