## 1. LU 分解

### 方法重点

- LU 分解将矩阵分解为 A = PLU（置换矩阵 × 下三角 × 上三角）。
- `linalg.lu(A)` 返回 `(P, L, U)` 三个矩阵。
- LU 分解是高斯消元法的矩阵形式，常用于高效求解线性方程组。
- P 是置换矩阵，用于行交换以保证数值稳定性。

### 参数速览（本节）

适用 API：`linalg.lu(A)`

| 参数名 | 本例取值 | 说明 |
|---|---|---|
| `A` | `[[1,2,3],[4,5,6],[7,8,10]]` | 待分解的方阵 |

### 示例代码

```python
import numpy as np
from scipy import linalg

A = np.array([[1, 2, 3], [4, 5, 6], [7, 8, 10]])

P, L, U = linalg.lu(A)

print(f"原矩阵 A:\n{A}")
print(f"\nP (置换矩阵):\n{P}")
print(f"\nL (下三角):\n{np.round(L, 4)}")
print(f"\nU (上三角):\n{np.round(U, 4)}")
print(f"\n验证 P @ L @ U:\n{np.round(P @ L @ U, 4)}")
```

### 结果输出

```text
原矩阵 A:
[[ 1  2  3]
 [ 4  5  6]
 [ 7  8 10]]

P (置换矩阵):
[[0. 0. 1.]
 [0. 1. 0.]
 [1. 0. 0.]]

L (下三角):
[[1.     0.     0.    ]
 [0.5714 1.     0.    ]
 [0.1429 0.5    1.    ]]

U (上三角):
[[7.     8.     10.    ]
 [0.     0.4286 0.2857]
 [0.     0.     0.5   ]]

验证 P @ L @ U:
[[ 1.  2.  3.]
 [ 4.  5.  6.]
 [ 7.  8. 10.]]
```

### 理解重点

- P 将第 3 行移到第 1 行（选主元策略），保证 L 的元素绝对值 ≤ 1。
- L 的对角线全为 1（单位下三角矩阵），U 的对角线是主元。
- P @ L @ U 精确重构了原矩阵 A，验证分解正确性。
- LU 分解一次后可以高效求解多个右端向量的线性方程组。

## 2. QR 分解

### 方法重点

- QR 分解将矩阵分解为 A = QR（正交矩阵 × 上三角矩阵）。
- Q 的列向量是正交的（Q^T Q = I），R 是上三角矩阵。
- QR 分解适用于非方阵，常用于最小二乘问题和特征值算法。
- `linalg.qr(A)` 默认返回完整 QR 分解。

### 参数速览（本节）

适用 API：`linalg.qr(A)`

| 参数名 | 本例取值 | 说明 |
|---|---|---|
| `A` | `[[1,2],[3,4],[5,6]]` | 待分解的 3×2 矩阵 |

### 示例代码

```python
import numpy as np
from scipy import linalg

A = np.array([[1, 2], [3, 4], [5, 6]])

Q, R = linalg.qr(A)

print(f"原矩阵 A (3x2):\n{A}")
print(f"\nQ (正交矩阵):\n{np.round(Q, 4)}")
print(f"\nR (上三角):\n{np.round(R, 4)}")
print(f"\n验证 Q @ R:\n{np.round(Q @ R, 4)}")
```

### 结果输出

```text
原矩阵 A (3x2):
[[1 2]
 [3 4]
 [5 6]]

Q (正交矩阵):
[[-0.1690  0.8971  0.4082]
 [-0.5071  0.2760 -0.8165]
 [-0.8452 -0.3451  0.4082]]

R (上三角):
[[-5.9161 -7.4370]
 [ 0.0000  0.8281]
 [ 0.0000  0.0000]]

验证 Q @ R:
[[1. 2.]
 [3. 4.]
 [5. 6.]]
```

### 理解重点

- A 是 3×2 矩阵，Q 是 3×3 正交矩阵，R 是 3×2 上三角矩阵。
- Q 的列向量两两正交且模为 1（可通过 Q^T @ Q 验证）。
- R 的前 2 行是上三角，第 3 行全零（因为 A 只有 2 列）。
- QR 分解是 Gram-Schmidt 正交化过程的矩阵形式。

## 3. SVD 分解

### 方法重点

- SVD（奇异值分解）将矩阵分解为 A = UΣVᴴ。
- U 和 V 是正交矩阵，Σ 是对角矩阵，对角元素为奇异值（非负且递减）。
- SVD 适用于任意形状的矩阵，是最通用的矩阵分解方法。
- 奇异值反映矩阵的"信息量"，用于降维、压缩和伪逆计算。

### 参数速览（本节）

适用 API：`linalg.svd(A)`

| 参数名 | 本例取值 | 说明 |
|---|---|---|
| `A` | `[[1,2,3],[4,5,6]]` | 待分解的 2×3 矩阵 |

### 示例代码

```python
import numpy as np
from scipy import linalg

A = np.array([[1, 2, 3], [4, 5, 6]])

U, s, Vh = linalg.svd(A)

print(f"原矩阵 A (2x3):\n{A}")
print(f"\nU:\n{np.round(U, 4)}")
print(f"\n奇异值 s: {np.round(s, 4)}")
print(f"\nVh:\n{np.round(Vh, 4)}")

# 重构
S = np.zeros_like(A, dtype=float)
S[:len(s), :len(s)] = np.diag(s)
reconstructed = U @ S @ Vh
print(f"\n重构 U @ S @ Vh:\n{np.round(reconstructed, 4)}")
```

### 结果输出

```text
原矩阵 A (2x3):
[[1 2 3]
 [4 5 6]]

U:
[[-0.3863 -0.9224]
 [-0.9224  0.3863]]

奇异值 s: [9.5080 0.7729]

Vh:
[[-0.4287 -0.5663 -0.7039]
 [ 0.8060  0.1124 -0.5812]
 [ 0.4082 -0.8165  0.4082]]

重构 U @ S @ Vh:
[[1. 2. 3.]
 [4. 5. 6.]]
```

### 理解重点

- `linalg.svd` 返回的 `s` 是一维数组（奇异值），需要手动构造对角矩阵 Σ。
- 奇异值 9.51 >> 0.77，说明矩阵的主要信息集中在第一个奇异值对应的方向。
- 保留前 k 个最大奇异值可实现矩阵的低秩近似（数据压缩）。
- U @ S @ Vh 精确重构原矩阵，验证了分解的正确性。

## 4. 特征值与特征向量

### 方法重点

- `linalg.eig(A)` 计算方阵的特征值和特征向量。
- 特征方程 Av = λv：矩阵乘以特征向量等于特征值缩放特征向量。
- 特征值可能是复数（即使矩阵元素全为实数）。
- 特征向量按列排列在返回的矩阵中，第 i 列对应第 i 个特征值。

### 参数速览（本节）

适用 API：`linalg.eig(A)`

| 参数名 | 本例取值 | 说明 |
|---|---|---|
| `A` | `[[4,2],[1,3]]` | 待分解的方阵 |

### 示例代码

```python
import numpy as np
from scipy import linalg

A = np.array([[4, 2], [1, 3]])

eigenvalues, eigenvectors = linalg.eig(A)

print(f"矩阵 A:\n{A}")
print(f"\n特征值: {eigenvalues}")
print(f"\n特征向量:\n{eigenvectors}")

# 验证 A @ v = λ * v
print("\n验证 A @ v = λ * v:")
for i in range(len(eigenvalues)):
    v = eigenvectors[:, i]
    lam = eigenvalues[i]
    lhs = A @ v
    rhs = lam * v
    print(f"  λ={lam:.4f}: A@v = {np.round(lhs, 4)}, λ*v = {np.round(rhs, 4)}")
```

### 结果输出

```text
矩阵 A:
[[4 2]
 [1 3]]

特征值: [5.+0.j 2.+0.j]

特征向量:
[[ 0.8944 -0.7071]
 [ 0.4472  0.7071]]

验证 A @ v = λ * v:
  λ=5.0000+0.0000j: A@v = [4.4721 2.2361], λ*v = [4.4721 2.2361]
  λ=2.0000+0.0000j: A@v = [-1.4142  1.4142], λ*v = [-1.4142  1.4142]
```

### 理解重点

- 矩阵 A 的特征值为 5 和 2（特征方程 det(A-λI)=0 → λ²-7λ+10=0）。
- A@v = λ*v 验证成功：矩阵作用在特征向量上，只改变长度不改变方向。
- 特征值以复数形式返回（即使是实数，也带 `+0.j`），使用 `.real` 提取实部。
- 几何意义：特征向量是矩阵变换下方向不变的"轴"，特征值是沿该轴的缩放因子。

## 5. 线性方程组求解

### 方法重点

- `linalg.solve(A, b)` 直接求解 Ax = b，比 `np.linalg.inv(A) @ b` 更高效和数值稳定。
- 内部使用 LU 分解实现，时间复杂度 O(n³)。
- 要求 A 是方阵且非奇异（行列式不为零）。
- 对于超定方程组（m > n），应使用 `linalg.lstsq` 求最小二乘解。

### 参数速览（本节）

适用 API：`linalg.solve(A, b)`

| 参数名 | 本例取值 | 说明 |
|---|---|---|
| `A` | `[[3,1],[1,2]]` | 系数矩阵 |
| `b` | `[9, 8]` | 右端向量 |

### 示例代码

```python
import numpy as np
from scipy import linalg

# Ax = b
A = np.array([[3, 1], [1, 2]])
b = np.array([9, 8])

x = linalg.solve(A, b)

print("方程组:")
print("  3x + y = 9")
print("  x + 2y = 8")
print(f"\n解: x = {x[0]:.4f}, y = {x[1]:.4f}")
print(f"验证 A @ x = {A @ x}")
```

### 结果输出

```text
方程组:
  3x + y = 9
  x + 2y = 8

解: x = 2.0000, y = 3.0000
验证 A @ x = [9. 8.]
```

### 理解重点

- 方程组的解为 x=2, y=3，代入验证：3×2+3=9, 2+2×3=8，正确。
- `linalg.solve` 比求逆矩阵再相乘更快（避免显式求逆），数值更稳定。
- 验证 A @ x = b 是检验解正确性的标准方法。
- 对于大规模稀疏方程组，应使用 `scipy.sparse.linalg.spsolve`。

## 常见坑

| 坑 | 说明 |
|---|---|
| `eig` 返回复数 | 即使矩阵是实数，特征值也以复数形式返回，需要 `.real` 提取 |
| `svd` 的 s 是一维数组 | 不是对角矩阵，重构时需要手动构造 Σ 矩阵 |
| `solve` 要求方阵 | A 必须是方阵，非方阵用 `lstsq` |
| `scipy.linalg` vs `numpy.linalg` | SciPy 版本功能更多（如 LU 分解），且有时更快 |
| 数值精度 | 矩阵条件数大时，分解和求解的精度会下降 |

## 小结

- LU 分解将矩阵分解为置换矩阵 × 下三角 × 上三角，用于高效求解线性方程组。
- QR 分解产生正交矩阵和上三角矩阵，用于最小二乘和特征值算法。
- SVD 是最通用的矩阵分解，奇异值反映矩阵的信息结构，用于降维和压缩。
- 特征值分解揭示矩阵的内在几何特性，特征向量是变换的不变方向。
- `linalg.solve` 是求解线性方程组的首选方法，比求逆矩阵更高效和稳定。
